# 介绍

## FlowMind 是什么

FlowMind 是一个 **LLM 驱动的智能项目管理平台**：在看板协作的基础上，深度集成 RAG 知识库与 AI 助手，让 AI 真正读懂你的项目——它既能检索你上传的文档，也能直接操作看板任务。

![登录页](/images/01-login.png)

## 核心能力

| 能力 | 说明 |
|------|------|
| 📋 智能看板 | 拖拽任务卡片、自定义状态列、子任务、列内筛选与排序 |
| 🤖 AI 助手 | 自然语言操作看板，SSE 流式输出，工具调用过程可见 |
| 📚 RAG 知识库 | 文档自动解析索引，向量 + 关键词 RRF 混合检索 |
| 🌐 跨项目助手 | 一次提问覆盖你参与的所有项目，结果标注来源 |
| ⚙️ 配置中心 | LLM / Embedding 在线配置，免重启即时生效 |
| 🔐 权限安全 | 注册审批、项目角色分层、JWT 鉴权、越权返回 404 |

## 设计理念

- **AI 不是外挂，是协作成员**：助手通过工具调用直接读写看板，操作可撤销、可追溯。
- **无命中不编造**：知识库检索设有相似度阈值，查不到就如实说明，不胡编答案。
- **配置不出安全事故**：密钥脱敏展示、来源可追溯（环境变量 / 运行时覆盖），超管在线改配置免重启。
- **渐进可用**：没有 LLM Key 也能跑——知识库自动降级为关键词检索，其余功能不受影响。
