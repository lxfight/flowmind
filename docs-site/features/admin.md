# 系统管理

## 超管配置中心

超级管理员进入 **系统配置** 页，可在线管理平台的核心配置：

![系统配置](/images/07-admin-config.png)

- **LLM 配置**：API Key、Base URL、对话模型，支持连通性测试
- **Embedding 配置**：独立于 LLM 的 Key 与地址，独立连通性测试
- **检索参数**：相似度阈值等知识库行为调优
- **免重启生效**：保存即写入运行时配置，无需重启服务

密钥脱敏展示，每项配置标注来源（环境变量 / 运行时覆盖）。详见[配置说明](/guide/configuration)。

## 用户与权限

| 角色 | 能力 |
|------|------|
| 超级管理员 | 系统配置、用户审批、全部项目管理 |
| 项目成员 | 看板协作、知识库、AI 助手 |
| 项目观察者 | 只读访问 |

- **注册审批**：新用户注册后需超管审批方可登录，防止开放注册被滥用
- **成员管理**：项目内可查看候选用户列表，一键添加成员

![成员页](/images/06-members.png)

## 安全设计

- JWT 鉴权 + bcrypt 密码加密
- 登录防暴力破解（失败次数限制）
- 越权访问一律返回 **404**，不泄露资源存在性
- 会话与数据按用户、项目严格隔离
