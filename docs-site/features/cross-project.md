# 跨项目助手

「我的项目」页同样内置了 AI 助手入口（右下角浮动按钮）。与看板页的"项目内助手"不同，它的视野覆盖**你参与的所有项目**。

![我的项目页](/images/02-dashboard.png)

## 一次提问，全局检索

- 任务、知识库检索同时覆盖全部参与项目
- 检索结果**标注来源项目名**，答案的出处一目了然
- 典型问题：「我所有项目里有哪些任务快到期了？」「汇总一下各个项目的进度」

## 写操作自动追问

跨项目语境下，写操作必须落到具体项目。当你说"创建一个任务"时，助手不会乱猜，而是通过**追问卡片**列出候选项目，点选确认后才执行——与项目内助手共用同一套追问机制。

## 会话归属

- 跨项目会话独立于任何单项目会话，按用户隔离
- 会话列表与看板页互不干扰：项目页只显示该项目会话，「我的项目」页只显示跨项目会话

::: info @ 提及
跨项目模式下成员横跨多个项目，聊天输入框的 @ 补全暂不启用；评论中的 @ 提及不受影响。
:::
