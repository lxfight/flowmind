# 智能看板

看板是 FlowMind 的协作中心：任务以卡片形式在自定义状态列间流转，支持拖拽、子任务、评论与实时通知。

![智能看板](/images/03-kanban.png)

## 核心能力

- **拖拽式任务卡片**：@dnd-kit 驱动，列间拖动即时落库
- **自定义状态列**：增删改状态列，按团队流程自由编排
- **任务要素齐全**：子任务拆分、优先级、截止日期、负责人、评论
- **评论 @ 提及**：被 @ 的成员实时收到通知

## 列内筛选与排序

看板顶部工具栏提供纯前端过滤（任务已全量加载，零延迟）：

- **关键词筛选**：匹配标题与描述
- **负责人筛选**：按成员过滤
- **优先级筛选**：只看高优 / 中 / 低

每列内可独立排序：

- 按创建时间 / 更新时间 / 优先级 / 截止日期
- 升降序一键切换

::: tip
筛选条件保存在组件状态，刷新页面不保留——看板始终回到全局视图，避免"找不到任务"的困惑。
:::
