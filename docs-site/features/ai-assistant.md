# AI 助手

每个看板页右下角的浮动按钮打开 AI 助手面板。助手不是"只会聊天的外挂"——它通过**工具调用**直接读写你的项目。

![AI 助手面板](/images/04-chat-panel.png)

## SSE 流式对话

- 回复逐字流出（SSE），状态消息实时更新：「正在搜索任务…」「🔧 正在创建任务…」
- 工具调用的开始与结束都有事件，过程完全透明
- 可随时点击停止，中断当前生成

## 助手能做什么

助手通过一组工具与项目交互，常见操作一句话完成：

| 能力 | 示例 |
|------|------|
| 查询任务 | 「这个项目有哪些任务？」「高优先级的任务还剩几个？」 |
| 创建任务 | 「帮我创建一个任务：整理本周迭代计划」 |
| 调整任务 | 「把『接口联调』移到进行中，截止明天」 |
| 知识库问答 | 「根据需求文档，拆一下登录模块的开发任务」 |
| 总结进度 | 「总结一下当前项目进度」 |

写操作（创建 / 更新 / 移动任务）按批次记录，支持**一键撤销**。

## @ 提及成员

聊天输入框键入 `@` 触发项目成员自动补全（键盘可选）。被 @ 的成员会收到通知——AI 对话同样融入了团队的沟通链路。

## 会话隔离

- 会话按 **用户 × 项目** 严格隔离：同一项目里你看不到别人的会话，切换项目只显示当前项目的会话
- 会话支持重命名、删除；每条会话的上下文独立

## 追问机制

当指令缺少必要信息（例如"要写到哪个项目"），助手不会瞎猜，而是通过追问卡片让你点选确认，回答后继续执行。
